
## Installation

```bash
$ npm install
```

## Running the app

create the .env file refereeing to example.env

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

import the postman collection from campaign.postman_collection.json and test the endpoints

deployed at https://testnets.opensea.io/collection/campaignnft
